# DeepAgents 适用场景

## 核心适用场景

### 1. 复杂研究
- 文献综述、数据分析、报告生成
- 深度调研、行业分析、竞品研究

### 2. 软件开发
- 全栈项目生成、测试、部署
- 代码编写、调试、重构
- 技术文档生成

### 3. 内容创作
- 长文、小说、剧本、营销文案
- 技术博客、白皮书、产品文档

### 4. 企业自动化
- 报表生成、流程审批、数据处理
- 跨系统任务编排、日志分析、运维自动化

## 典型工作流程示例

以写一篇技术博客为例：

1. **规划**：生成待办（调研→大纲→初稿→修改→发布）
2. **子 Agent 分工**：
   - Researcher：搜索资料、整理要点
   - Writer：写初稿、保存到 VFS
   - Reviewer：修改润色
3. **上下文管理**：长文本写入文件，避免窗口溢出
4. **执行与迭代**：按计划执行，失败调整待办
5. **交付**：汇总结果，输出最终博客

## 不适配场景

- 简单问答场景（过于重量级）
- 需要高度定制化流程控制（建议直接用 LangGraph）
- 轻量级原型开发（建议用普通 LangChain Agent）
