# AgentScope 适用场景

## 核心适用场景

### 1. 产业落地
- 菜鸟物流仓储调度
- 盒马生鲜供应链智能决策
- 库存盘点、订单分拣流程优化

### 2. 企业数字化
- 财务单据审核智能体集群
- 市场数据分析智能体
- 报表自动生成与行业数据研判

### 3. 科研场景
- 仿真多智能体环境搭建
- 社会学模拟实验
- 大模型智能体能力评测

### 4. 轻量化应用
- 文档总结助手
- 代码辅助编程
- 多角色对话机器人

### 5. 需要灵活多 Agent 编排的场景
- 动态任务交接
- 多角色智能体自主分工
- 复杂工作流编排

### 6. 需要本地部署或私有化的场景
- K8s 集群部署
- 私有化部署需求

### 7. 需要特定领域微调的场景
- 内置微调支持
- 领域优化 Agent 行为

## 典型应用示例

以供应链智能决策为例：

1. **智能体分工**：采购智能体、库存智能体、配送智能体、销售预测智能体
2. **消息交互**：通过 Message Hub 实现智能体间动态任务交接
3. **记忆管理**：利用长期记忆保存历史数据，支持趋势分析
4. **实时干预**：关键决策点人工审批，确保业务安全

## 不适配场景

- 需要严格流程控制（建议选 LangGraph/CrewAI）
- 只需要 API 调用不想管基础设施（建议用 LangChain）
- 个人开发原型（AgentScope 稍重）
