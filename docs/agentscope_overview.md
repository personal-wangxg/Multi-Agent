# AgentScope 概述

## 定位

AgentScope 是**阿里通义实验室推出的生产级多智能体一站式开发框架**，聚焦解决智能体全生命周期管理痛点，完整覆盖从需求开发、环境部署到线上运维监控的全流程。

## 核心设计哲学

**释放优先（capability-first）**：充分发挥模型自己的推理和工具调用能力，不在框架层做过度的流程强制，让模型自己决定怎么用。

## 关键属性

- **开源协议**：Apache 2.0（商用友好）
- **模型中立**：不绑定任意大模型，支持通义千问、Qwen、GPT、Claude、GLM等
- **全生命周期**：开发、部署、监控一体化架构

## 三层技术架构

| 层级 | 组件 | 职责 |
|-----|------|-----|
| 核心框架 | AgentScope Core | 智能体构建与应用编排 |
| 运行时 | AgentScope Runtime | 安全可靠的运行和部署环境 |
| 可视化 | AgentScope Studio | 可视化开发与监控工具 |

## 与其他框架对比

| 框架 | 编排哲学 | 模型要求 |
|-----|---------|---------|
| AgentScope 2.0 | 弱编排，释放模型能力 | 越强越好 |
| LangChain/LangGraph | 强编排，流程优先 | 中等 |
| CrewAI/AutoGen | 强编排，角色固定 | 中等 |
