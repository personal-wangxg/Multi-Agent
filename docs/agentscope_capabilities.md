# AgentScope 核心能力

## 核心组件

### 1. 消息驱动的通信中枢

- `msghub` 模块实现智能体间消息的统一管理与路由
- 支持多模态数据传输（文字、图片、语音）
- 通过 URL 解耦存储与传输
- 通信过程对开发者透明可见

### 2. 智能体构建与任务编排

- 多层次智能体构建接口
- YAML 配置文件实现低代码开发
- Python 接口深度定义思考逻辑与行为模式
- 原生支持多样化协作模式：
  - 路由式任务分发
  - 并行化处理
  - 协调者-工作者架构
  - 评估者-优化者闭环

### 3. 模块化组件与可插拔设计

#### 模型接口
- `ModelWrapper` 抽象统一接入逻辑
- 支持 17+ LLM API 提供商及本地模型服务
- "一次编程，多模型运行"

#### 记忆系统
- 短期记忆优化：动态压缩、混合压缩技术
- 跨会话长期记忆：动态/静态/混合三种模式
- 上下文管理重构，终结"失忆"、"混乱"与"归零重启"

#### 工具调用
- 标准化工具封装接口
- 支持 MCP 协议
- 高效的工具调用能力

### 4. 实时介入控制

- 安全中断，支持状态持续
- 实时打断，支持用户实时干预
- 灵活定制，支持自定义中断处理逻辑

### 5. 多环境部署

- **本地部署**：适合开发调试
- **Serverless 云端**：无需管理基础设施
- **K8s 集群**：生产级大规模部署

### 6. 可观测性与工程化

- 内置 OTel（OpenTelemetry）支持
- 运行日志采集、性能监控、异常报错捕获
- Token 成本统计、故障重试机制
- 全链路追踪

### 7. 模型微调支持

- 框架内置模型微调支持
- 适合需要在特定领域优化 Agent 行为的团队
