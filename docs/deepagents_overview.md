# DeepAgents 概述

## 定位

DeepAgents 是 **LangChain 官方推出的生产级 Agent Harness（智能体驾驭层）**，基于 **LangGraph（执行引擎）+ LangChain（工具/模型层）** 构建，面向**长周期、多步骤、复杂目标**设计。

## 核心定位

将 LLM 的 "智能" 包装成**可观测、可控制、可稳定运行**的复杂任务执行环境。

## 关键属性

- **开源协议**：MIT（商用自由）
- **模型无关**：支持所有工具调用型 LLM
- **运行时**：LangGraph 提供持久化、断点续跑、流式输出

## 生态位置

在 LangChain 生态中位于高层封装：

| 层级 | 组件 | 定位 |
|-----|------|-----|
| 上层 | DeepAgents | 开箱即用的重型 Agent 构建器 |
| 中层 | LangChain | 应用框架与组件库 |
| 底层 | LangGraph | 图编排运行时引擎 |

## 一句话对比

- **普通 Agent**：你问一句、它做一步（临时工）
- **DeepAgents**：你给一个目标，它拆计划、分工、执行、沉淀、交付（项目负责人）
